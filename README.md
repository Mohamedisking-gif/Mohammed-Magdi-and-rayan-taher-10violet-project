# ♟️ Chess vs Ahmed 👨‍🏫

A fun and interactive chess game built using JavaScript, where you play against a smart AI opponent with a unique twist — your opponent is a strict teacher who reacts to your moves with voice feedback.

---

#🚀 Features

* ♟️ Play chess against an AI (3 difficulty levels)
* 🧠 Smart AI using Minimax algorithm
* 🎯 Click-based move highlighting
* 🔊 Voice feedback (teacher-style reactions)
* 🎨 Clean Chess.com-inspired interface
* 🎬 Win/Lose GIF animations
* 💀 Special "Final Exam" mode with intense effects

--



---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Chess.js (game logic)
* Chessboard.js (UI board)

---

## 📂 Project Structure

* `index.html` → Main game page
* `style.css` → Styling and layout
* `script.js` → Game logic and AI
